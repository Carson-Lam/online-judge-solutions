# Dice Roller

## Problem Description
Simulate a simple dice game where two players each roll a dice with the goal of achieving a sum of 7 with the two dice. If neither players achieve a sum of 7, the player with a larger total wins. If both players achieve the same sum, the game is declared a draw.

## Input Specifications
N/A

## Output Specifications
Output the results of both dice rolls as well as the result of the game.
